3. CSS (Cascading Style Sheets)

			[기본구조]
			

			

				 
				· Selector (선택자)
				
				
				· Property (속성)
				· Value (값)
				
				 
				:<article> {
				  지정할 스타일 내용
				}
				:지정할 내용들 (font-size, color position 등)
				:속성에 할당 될 내용들 (16px, blue, '굴림체' 등)
				
			

			
	
			   *속성과 값을 한 쌍으로 이 한 쌍을 "(Declaration)선언"이라고 칭함.
			   *각 선언은 "(세미콜론);"으로 구분.
			   예) article { font-size: 10px; color: white;  }

			[가변 크기 단위]
			

			

				 
				· em
				· %
				· rem
				· vw
				· vh
				· vmin
				· vmax
				· ex
				
				 
				:배수 단위 (상위)
				:백분율 단위
				:root me (최상위)
				:화면빈 백분율 단위
				:화면비 백분율 단위
				:화면비 최솟값
				:화면비 최댓값
				:현재 폰트의 x-height값
				
				
			

			

			   *반응형 구현을 위해 rem 사용 권장
			[색상 표현]
			

			

				 
				· 키워드
				· hex코드
				· hex코드에 Alpha값 사용
				· RGB
				· RGB에 Alpha값 사용
				
				 
				:blue, red 등
				:#E10034
				:#E1003444
				:rgb(124,123,111)
				:rgba(222,111,23, 0.4)
				
			

			


			[단순 선택자]
			

			

				 
				· 타입 선택자
				· 아이디 선택자
				
				· 클랙스 선택자
				
				· 전체 선택자
				· 속성 선택자
				
				
				
				 
				:지정한 태그를 사용하는 모든 곳에 적용
				:특정 아이디에만 적용(아이디 중복 비추) 
				 예) #id { color: red; }
				:특정 클래스에만 적용(한 클래스에 여러 요소 포함)
				 예) .ClassName { color: red; }
				:모든 요소에 적용 ( * )
				:특정 속성에만 적용
				 예1) a[target="_blank"] { color: red; }
				 예2) .ClassName[target="_blank"] { color: red; }
				
			

			


			[복합 선택자]
			

			

				 
				· 부모 - 자식 관계
				· 부모 - 후손 관계
				
				 
				:'>'등호를 사용하여 관계 표시
				:' '공백을 사용하여 관계 표시
				
			

			

			[box개념]
			

			

				 
				· 박스단위로 글자 or 그림등의 콘텐츠가 작성됨.
				· margin, border, padding 개념 알기
				· 가운데는 content
				
				
			

			

			[다양한 텍스트 스타일]
			

			

				 
				· font-size
				· font-family
				
				· font-style (font-weight)
				
				· 한번에 작성 가능
				· text-align
				· line-height
				· letter-spacing
				· text-indent
				
				 
				:글꼴 크기
				:글꼴 종류
				 글꼴은 기본 글꼴 or 웹폰트 or 업로드하여 사용가능.
				:글꼴 형태 (굵기 normal/bold or 100~900)
				 normal, italic, oblique(강제기울림)
				:font: font-style font-weight size font-family;
				:정렬하기 (left, right, center, justify)
				:줄간격 ( n배 or 30px )
				:자간 ( 10px )
				:들여쓰기 ( 10px )
